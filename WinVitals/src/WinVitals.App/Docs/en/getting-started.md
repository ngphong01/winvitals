# WinVitals User Guide
## Quick Start
1. Dashboard -> system overview
2. Scan -> START SCAN -> GO TO CLEAN
3. Performance -> CPU/RAM/Disk charts

## Scan Presets
Quick: temp, cache, logs, crash dumps
Deep: Quick + Windows Update cache, old installers
Developer: node_modules, build output, Python cache

## Clean Flow
Preset -> Preview -> Confirm (5s countdown)
High Risk files go to Quarantine (14 days)

## Shortcuts
Ctrl+1-8: Navigate | Ctrl+K: Commands | F1: Help
